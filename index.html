What is the purpose of the doctype declaration?

Frankly and simply speaking, the doctype declaration is simply there to tell the browser
what specific HTML version are we going to use to render the documents, in which it
usually will be <!DOCTYPE html>

Well, how about an HTML element, what is that?

Simply, an HTML element is the root element of the html document and anything that
follows it are its descendants. It is what basically anchors the primary structure and
content of our html document in which the other elements would then follow.

What is the purpose of the head element?

The head element or better seen as the <head> element is mainly the element that will contain metadata or meta-information pertaining to our HTML. This can range from the character encoding to utilize, connection to our CSS file and even the title of the webpage (through the following tags: <title>).

What is the purpose of the body element?

The body element or better seen as the <body> element is mainly the element that will contain the core content or of structured content and of assets we may put inside a webpage; akin to a body of an email or letter, it is where the main message of an email or letter is placed.
